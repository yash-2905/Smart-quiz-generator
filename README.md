
# Smart Quiz Generator (Django)

A web-based quiz application built using Django where users can generate quizzes and get instant scores.

---

##  Features

* Dynamic quiz generation
* Score calculation system
* Clean user interface
* Backend powered by Django

---

##  Tech Stack

* Python
* Django
* HTML, CSS

## Installation

```bash
git clone https://github.com/yash-2905/smart-quiz-generator.git
cd smart-quiz-generator
pip install -r requirements.txt
python manage.py runserver
```
